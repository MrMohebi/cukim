 ## ResOwner API
| address | Description  |
| --- |---   |
| [signup](#/resowner/signup) | register a new res owner |
| [createNewRes](#create-new-restaurant) | create a new restaurant |


> ### ```/resOwner/signup``` ```POST```
>
> #### Required fields:
>   - **username**
>   - **password**
>   - **name**
>
>   #### Return Values ``JSON``:
>   ```json
>   {
>     "statusCode": "[code]"
>   }
>   ```



### create new restaurant

> ```url: PUBLIC-DOMAIN/api/v1/resOwner/createNewRes``` ```POST```
> #### Required fields:
>   - **token**
>   - **username**
>   - **password**
>   - **persianName**
>   - **englishName**
>
>   #### Return Values ``JSON``:
>   ```json
>   {
>     "statusCode": "[code]"
>   }
>   ```
